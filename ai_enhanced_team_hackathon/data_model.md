# Data Model

## User Entity
- user_id (Primary Key, UUID): Unique user identification string.
- email (String, Required): User email address for identity management.
- created_at (Timestamp): Account registration date.

## InterviewSession Entity
- session_id (Primary Key, UUID): Unique simulation session tracking code.
- user_id (Foreign Key): Links back to the User entity.
- job_description (Text): Raw text of the target job position.

## QuestionAnswer Entity
- qa_id (Primary Key, UUID): Unique question-answer pair identifier.
- session_id (Foreign Key): Relates the record to an active InterviewSession.
- prompt_question (Text): The question generated by the AI system.
- user_response (Text): The textual answer provided by the candidate.
- evaluation_score (Integer): Calculated numerical evaluation grade.
