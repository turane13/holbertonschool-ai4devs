# Bug Descriptions

## bug1.py
- **Intended Behavior**: Return the last n items in a list.
- **Current Issue**: Off-by-one error due to incorrect slicing.

## bug2.js
- **Intended Behavior**: Calculate the total sum of price and tax as numbers.
- **Current Issue**: Performs string concatenation instead of mathematical addition.

## bug3.java
- **Intended Behavior**: Compare the input string safely with a literal string.
- **Current Issue**: Throws a NullPointerException because the variable is null.

## bug4.py
- **Intended Behavior**: Securely fetch user details from the database by username.
- **Current Issue**: Vulnerable to SQL Injection due to string formatting in the query.
