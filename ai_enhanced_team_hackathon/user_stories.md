# User Stories

## User Story 1
As a candidate, I want to paste a job description so that I can generate tailored interview questions.

## User Story 2
As a job seeker, I want to submit my answers textually so that the system can evaluate my performance.

## User Story 3
As a user, I want to view a detailed scoring matrix so that I can identify my technical weak points.

## User Story 4
As a student, I want to read suggested response templates so that I can learn how to improve my answers.

## User Story 5
As an interviewer, I want to configure the difficulty level so that the simulation matches candidate seniority.

## User Story 6
As a premium user, I want to export my interview history as a PDF report so that I can track progress over time.

## User Story 7
As an administrator, I want to monitor API usage metrics so that system prompt consumption stays within budget.

## User Story 8
As a system auditor, I want to access historical simulation logs so that data privacy rules remain compliant.
